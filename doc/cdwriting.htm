Multimedia classes

A collection of WPS classes for extending the MM subsystem of OS/2. They integrate the MMIO procedures for MP3 and OGG files written by Russel O'Connor http://www.math.berkeley.edu/~roconnor/ seamlessly into the WPS. A new media folder plays any audio class known to OS/2.

Features

  • Integrates MP3- and OGG-Files into the WPS
  • Audio details of any audio class in folder details view
  • Audio details of any audio class in the settings notebook
  • ID3 tags in the details view of folders
  • ID3 tag editing in the settings notebook of an MP3-file

http://www.os2world.com/cdwriting

MMIOMP3 and MMIOOGG

These are IO procedures which extend the multimedia subsystem of OS/2 so multimedia applications can read or play MP3 and OGG files.

http://www.math.berkeley.edu/~roconnor/

z!

z! is a great MP3 player for OS/2 which perfectly integrates with Audio-CD-creator. Get it for MP3 decoding and MP3 listening. It's far superior than the included mpg123 and it's free.
